# 📡 Документация по использованию бота AutoParseAlertBot

## 🚀 Основные функции

1. **Отслеживание ключевых слов** — мониторинг новых сообщений в указанных Telegram-группах и каналах в реальном
   времени.
2. **AI-поиск групп/каналов** — два режима поиска (локальный и глобальный) с использованием искусственного интеллекта (
   Groq, OpenRouter, G4F).
3. **Актуализация базы данных** — автоматическое обновление информации о типах (группа/канал) и ID площадок
   администратором.
4. **Проверка групп** — ручная проверка конкретных чатов на наличие ключевых слов без подписки.
5. **Экспорт данных** — выгрузка базы в формат Excel (.xlsx) с фильтрацией по категориям и типам чатов.
6. **Многоязычная поддержка** — полный перевод интерфейса на русский и английский языки (Fluent i18n).
7. **Управление аккаунтами** — подключение нескольких аккаунтов через .session файлы для распределения нагрузки.
8. **Массовое добавление групп** — загрузка списков площадок через .txt файлы.
9. **Определение активности** — автоматический расчет статуса группы (active/inactive) по дате последнего сообщения.
10. **Интеллектуальная категоризация** — автоматическое присвоение тематических категорий группам с помощью AI.
11. **Определение языка** — автоматическое определение основного языка контента в группах.

## 🛠 Установка и настройка

### Быстрая установка

1. Запустите `setup.py` для автоматической установки зависимостей и первичной настройки:
   ```bash
   python setup.py
   ```

### Ручная установка

1. Установите необходимые библиотеки:
   ```bash
   pip install -r requirements.txt
   ```

2. Создайте файл `.env` в корневом каталоге проекта:
   ```env
   BOT_TOKEN=ваш_токен_бота
   ID=ваш_api_id
   HASH=ваш_api_hash
   GROQ_API_KEY=ваш_groq_api_key
   OPENROUTER_API_KEY=ваш_openrouter_api_key
   USER=прокси_пользователь
   PASSWORD=прокси_пароль
   IP=прокси_ip
   PORT=прокси_порт
   LANGUAGE=ru
   ```

3. Запустите бота:
   ```bash
   python main.py
   ```

## 🎯 Инструкция по использованию

### 1. Подключение аккаунта

Для работы (парсинга и поиска) боту нужны рабочие аккаунты:

1. Зайдите в **"⚙️ Настройки"** -> **"🔐 Подключить аккаунт"**.
2. Отправьте файл `.session` (Telethon).
3. Бот проверит валидность и добавит аккаунт в пул.

### 2. Настройка мониторинга

1. **Ключевые слова:** В настройках выберите **"🔍 Ввод ключевого слова"** и отправьте список слов/фраз (каждая с новой
   строки).
2. **Целевая группа:** Выберите **"📤 Подключить группу для сообщений"** и укажите `@username` группы, куда бот будет
   слать отчеты.
3. **Площадки:** Отправьте список `@usernames` или `.txt` файл со ссылками для мониторинга.

### 3. AI Поиск

При входе в меню **"✨ Поиск через AI"** вы увидите описание режимов:

- **Локальный поиск:** Использует AI для генерации вариаций запросов и ищет совпадения в Telegram.
- **Глобальный поиск:** Масштабный перебор запросов через все подключенные аккаунты.

### 4. Панель администратора

Доступна по кнопке **"🛡️ Панель администратора"**:

- **🏷️ Присвоить категорию:** Обработка базы через Groq или OpenRouter для распределения по темам.
- **🌐 Присвоить язык:** Определение языка (ru/en и др.) для каждой группы.
- **🔄 Актуализация БД:** Проверка доступности групп и обновление их метаданных.

## 🟢 Статусы активности

| Статус          | Описание                                                 |
|:----------------|:---------------------------------------------------------|
| 🟢 **active**   | Сообщения были в течение последних 30 дней               |
| 🔴 **inactive** | Группа заброшена (более 30 дней тишины)                  |
| ⚪ **unknown**   | Не удалось получить данные (приватность или ограничения) |

## 📁 Структура проекта

- `account_manager/` — логика работы с Telegram API (Telethon).
- `ai/` — интеграции с LLM (Groq, OpenRouter).
- `database/` — модели Peewee и работа с SQLite.
- `handlers/` — логика команд и диалогов aiogram.
- `locales/` — файлы локализации (.ftl).
- `keyboards/` — сборка кнопок интерфейса.
