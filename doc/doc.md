# 📡 Документация по использованию бота

## 🚀 Основные функции

1. Отслеживание ключевых слов в указанных Telegram-группах и каналах
2. Поиск групп/каналов по ключевым словам с помощью ИИ
3. Автоматическая 🔄 Актуализация базы данных Telegram-групп и каналов администратором
4. Проверка группы на наличие ключевых слов.
5. Экспорт данных в Excel формате
6. Многоязычная поддержка (русский/английский)
7. Получение логов работы бота через кнопку "📄 Получить лог файл" в панели администратора
8. Полный экспорт базы данных Telegram-групп и каналов в формате Excel
9. Разделённый экспорт базы данных по типам чатов (все группы/каналы, только каналы, только супергруппы, только обычные
   чаты)

Для самостоятельного запуска бота см. раздел ["🛠 Установка и настройка"](#установка-и-настройка) ниже.\
Для обычного пользователя — см. раздел ["🧭 Основное меню"](#основное-меню).

## 🛠 Установка и настройка

1. Запустите `setup.py` для установки зависимостей и создания `.env` файла
2. Заполните `.env` файл:
   BOT_TOKEN=ваш_токен_бота
   ID=ваш_id_приложения
   HASH=ваш_hash_приложения
   GROQ_API_KEY=ваш_groq_api_key
   OPENROUTER_API_KEY=ваш_openrouter_api_key
   USER=ваш_прокси_пользователь
   PASSWORD=ваш_прокси_пароль
   IP=ваш_прокси_ip
   PORT=ваш_прокси_порт
   LANGUAGE=ru
3. Запустите бота: python main.py

## 🎯 Пошаговая Инструкция по использованию

### 1. Начало работы

- Запустите бот командой `/start`
- Выберите язык интерфейса (Русский / English)

### 2. Подключение аккаунта

- Перейдите в меню **"Настройки"**
- Нажмите кнопку **"🔐 Подключить аккаунт"**
- Отправьте файл сессии в формате `+79599999999.session`  
  *(Создаётся при первом подключении через Telethon)*

### 3. Административные функции (только для администраторов)

- Нажмите **"Панель администратора"** в главном меню
- Используйте кнопки:
    - **"📄 Получить лог файл"** — загрузка журнала событий
    - **"🔄 Актуализация базы данных"** — обновление типов и ID групп/каналов с проверкой их текущего статуса в Telegram

### 3. Настройка отслеживания

#### Добавление ключевых слов:

- В меню **"Настройки"** выберите **"🔍 Ввод ключевого слова"**
- Введите одно или несколько слов/фраз, каждое с новой строки

#### Добавление групп для отслеживания:

- В меню **"Настройки"** выберите **"🔁 Обновить список"**
- Отправьте ссылки на группы/каналы одним из способов:
  • По одной в сообщении в формате `@username`
  • В виде .txt файла — каждая ссылка в столбик (на новой строке) в формате `@username`

📌 Пример содержимого файла:
@group1
@channel1
@my_community

#### Настройка группы для пересылки сообщений:

- В меню **"Настройки"** выберите **"📤 Подключить группу для сообщений"**
- Отправьте `@username` группы, куда будут пересылаться сообщения  
  ⚠ Группа должна быть публичной (или бот должен быть добавлен в приватную).  
  ✅ Рекомендуется использовать техническую группу с неприметным названием (например, `@dfgdfgtrgrgrtgr`).

---

### 4. Запуск отслеживания

- Нажмите **"Запуск отслеживания"** в главном меню
- Если не указаны обязательные параметры:
    - Ключевые слова
    - Группы для отслеживания
    - Группа для пересылки сообщений  
      — бот выдаст уведомление с подсказкой и предложит доделать настройки.

- После запуска:
    - Бот начнёт мониторинг новых сообщений в указанных чатах
    - Все сообщения, содержащие ключевые слова, будут пересланы в указанную группу

- Для остановки нажмите **"🛑 Остановить отслеживание"**

---

### 5. Поиск групп/каналов с помощью ИИ

> 📊 На стартовом экране отображается счётчик найденных групп. Чем активнее пользователи, тем больше база данных!

#### Как это работает:

1. Выберите в главном меню: **"Получить базу"**
2. Выберите режим:
    - **"AI поиск"** — введите ключевое слово (например, `Python`)
3. Бот:
    - Передаёт слово в ИИ (Groq AI)
    - ИИ генерирует список релевантных тем, названий, тегов
    - Бот запрашивает у Telegram-аккаунта наличие групп по этим названиям
    - Найденные группы сохраняются в базу данных (`bot.db`) и отправляются пользователю

#### Режимы поиска:

- **"AI поиск"** — умный поиск через ИИ
- **"📥 Вся база"** — получение полного списка всех ранее найденных групп в виде Excel-файла
- **"📥 База каналов"** — экспорт только каналов
- **"📥 База групп"** — экспорт только супергрупп
- **"📥 Получить всю базу Обычных чатов (группы старого типа)"** — экспорт только обычных чатов

> 📥 Формат файла: `Вся_база.xlsx`
> Содержит: Название, Юзернейм, Описание, Участники, Категория, Тип, Ссылка, Дата добавления

## 🧩 Доступные команды и интерфейсные действия

### `/start`

**Описание:** Запускает бота и открывает главное меню.  
**Доступ:** Все пользователи

### `📄 Получить лог файл`

**Описание:** Отправляет текущий файл логов `logs/log.log` администратору бота.  
**Доступ:** Только администраторы  
**Формат ответа:** `log.log` — документ Telegram с HTML-капшном  
**Особенности:** Логи за последний день (ротация), полезны для диагностики и аудита

### `Инструкция по использованию`

**Описание:** Показывает краткую справку по функциям бота  
**Доступ:** Все пользователи

### `🔐 Подключить аккаунт`

**Описание:** Позволяет пользователю загрузить `.session` файл своего Telegram-аккаунта для авторизации бота. Необходимо
для доступа к закрытым группам и увеличения лимитов запросов.
**Доступ:** Все пользователи

### `Запуск отслеживания`

**Описание:** Запускает процесс мониторинга указанных групп и каналов на наличие заданных ключевых слов. Перед запуском
проверяется наличие всех необходимых настроек.
**Доступ:** Все пользователи

### `Проверка группы на наличие ключевых слов`

**Описание:** Функция в разработке. Планируется реализовать проверку конкретной группы на наличие сообщений с ключевыми
словами.
**Доступ:** В разработке

### `🔍 Список ключевых слов`

**Описание:** Отображает текущий список ключевых слов пользователя в формате Excel-файла (`keywords.xlsx`). Позволяет
контролировать, по каким словам ведётся поиск.
**Доступ:** Все пользователи

### `🌐 Ссылки для отслеживания`

**Описание:** Показывает список групп и каналов, которые в данный момент отслеживаются, в формате Excel-файла (
`channels.xlsx`).
**Доступ:** Все пользователи

### `Получить базу`

**Описание:** Открывает меню поиска групп и каналов по ключевым словам с помощью ИИ (Groq AI) или получения полной базы
данных.
**Доступ:** Все пользователи

### `Настройки`

**Описание:** Центр настройки всех параметров бота: ключевые слова, группы для отслеживания, техническая группа для
пересылки.
**Доступ:** Все пользователи

### `Панель администратора`

**Описание:** Интерфейс для администраторов с доступом к диагностике и обслуживанию бота.
**Доступ:** Только администраторы

## 🧑‍💻 Разработка и поддержка

Если вы разработчик:

- Модель базы данных: `TelegramGroup` (в `database/database.py`)
- Поиск реализован через `search_group_ai()` и `handle_ai_search()`
- Экспорт в Excel — в обработчиках `export_all_groups()`, `export_channels()`, `export_supergroups()`,
  `export_legacy_groups()`
- Все строки интерфейса локализуемы (поддержка i18n в будущем)
- Функции названы семантически правильно: `extract_clean_group_name`, `create_group_identifier`,
  `classify_telegram_chat`, `upsert_telegram_group`, `generate_excel_export`, `create_search_summary`,
  `register_ai_search_handlers`
- Для успешной подписки на русскоязычные группы и каналы рекомендуется использовать аккаунты, зарегистрированные в
  странах с русскоязычной аудиторией — таких, как Украина, Россия или Узбекистан, так как аккаунты из других регионов
  могут не приниматься или блокироваться системой.

- Папки:
    - `accounts/<user_id>` — Файлы сессий пользователей
    - `data/` — Служебные данные
    - `examples/` — Примеры файлов для импорта (ключевые слова, группы)
    - `exports/` — Экспортированные файлы Excel
    - `logs/` — Логи работы бота

> 🚀 Бот идеально подходит для сбора данных, анализа рынка, мониторинга конкурентов и автоматизации поиска в Telegram.

> ⚠ Устаревший файл `config.ini` был полностью заменён на `.env`. Все настройки теперь хранятся в `.env` файле.
