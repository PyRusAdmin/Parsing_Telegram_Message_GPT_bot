# 📡 Документация по использованию бота AutoParseAlertBot

## 🚀 Основные функции

1. **Отслеживание ключевых слов** — мониторинг новых сообщений в указанных Telegram-группах и каналах
2. **AI-поиск групп/каналов** — поиск по ключевым словам с помощью ИИ (Groq AI)
3. **Актуализация базы данных** — автоматическое обновление информации о группах администратором
4. **Проверка групп** — проверка групп на наличие ключевых слов
5. **Экспорт данных** — выгрузка в Excel формате с фильтрацией по категориям
6. **Многоязычная поддержка** — русский и английский интерфейс
7. **Управление аккаунтами** — подключение через .session файлы с конвертацией в StringSession
8. **Массовое добавление групп** — загрузка списка групп через .txt файл
9. **Определение активности** — автоматическая проверка даты последнего сообщения в группах
10. **Получение логов** — доступ к логам через панель администратора

## 🛠 Установка и настройка

### Быстрая установка

1. Запустите `setup.py` для автоматической установки зависимостей и создания `.env` файла:
   ```bash
   python setup.py
   ```

### Ручная установка

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Создайте `.env` файл в корне проекта:
   ```env
   BOT_TOKEN=ваш_токен_бота
   ID=ваш_api_id
   HASH=ваш_api_hash
   GROQ_API_KEY=ваш_groq_api_key
   OPENROUTER_API_KEY=ваш_openrouter_api_key
   USER=прокси_пользователь
   PASSWORD=прокси_пароль
   IP=прокси_ip
   PORT=прокси_порт
   LANGUAGE=ru
   ```

3. Запустите бота:
   ```bash
   python main.py
   ```

### Получение API ключей

- **BOT_TOKEN** — получите у [@BotFather](https://t.me/BotFather)
- **ID и HASH** — получите на [my.telegram.org](https://my.telegram.org)
- **GROQ_API_KEY** — получите на [console.groq.com](https://console.groq.com)

## 🎯 Пошаговая инструкция по использованию

### 1. Начало работы

1. Запустите бот командой `/start`
2. Выберите язык интерфейса (Русский / English)
3. Ознакомьтесь с главным меню

### 2. Подключение аккаунта

Для работы бота необходим подключенный Telegram-аккаунт:

1. Перейдите в меню **"⚙️ Настройки"**
2. Нажмите кнопку **"🔐 Подключить аккаунт"**
3. Отправьте файл сессии в формате `.session` (например, `+79599999999.session`)
4. Бот автоматически конвертирует его в StringSession и сохранит в базе данных

> ⚠️ **Важно:** Для успешной подписки на русскоязычные группы рекомендуется использовать аккаунты, зарегистрированные в
> странах с русскоязычной аудиторией (Украина, Россия, Узбекистан).

### 3. Настройка отслеживания

#### Добавление ключевых слов:

1. В меню **"⚙️ Настройки"** выберите **"🔍 Ввод ключевого слова"**
2. Введите одно или несколько слов/фраз, каждое с новой строки
3. Бот сохранит ключевые слова и предложит добавить группы

#### Добавление групп для отслеживания:

**Способ 1: Через меню**

1. В меню **"⚙️ Настройки"** выберите **"🔁 Обновить список"**
2. Отправьте ссылки на группы/каналы в формате `@username`

**Способ 2: Через файл (массовое добавление)**

1. В меню **"⚙️ Настройки"** выберите **"🔁 Обновить список"**
2. Отправьте `.txt` файл со списком групп:
   ```
   @group1
   @channel1
   @my_community
   ```

> ✅ Преимущества массового добавления: быстрая загрузка большого количества групп, защита от дубликатов.

#### Настройка группы для пересылки сообщений:

1. В меню **"⚙️ Настройки"** выберите **"📤 Подключить группу для сообщений"**
2. Отправьте `@username` группы, куда будут пересылаться найденные сообщения
3. ⚠️ Группа должна быть публичной или бот должен быть добавлен в приватную группу

### 4. Запуск отслеживания

1. Нажмите **"🚀 Запуск отслеживания"** в главном меню
2. Бот проверит наличие всех необходимых настроек:
    - Подключенный аккаунт
    - Ключевые слова
    - Группы для отслеживания
    - Группа для пересылки
3. Если всё настроено корректно, бот начнет мониторинг
4. Все сообщения с ключевыми словами будут автоматически пересылаться в указанную группу

> Для остановки нажмите **"🛑 Остановить отслеживание"**

### 5. Поиск групп/каналов с помощью ИИ

На стартовом экране отображается счётчик найденных групп в базе данных.

#### Как это работает:

1. Выберите в главном меню: **"📊 Получить базу"**
2. Выберите режим поиска:
    - **"AI поиск"** — введите ключевое слово (например, `Python`)
    - **"📥 Вся база"** — получение полного списка
    - **"📥 База каналов"** — только каналы
    - **"📥 База групп"** — только супергруппы
    - **"📥 База обычных чатов"** — группы старого типа

3. При AI-поиске:
    - Бот передаёт слово в Groq AI
    - ИИ генерирует список релевантных названий
    - Бот ищет группы через Telegram API
    - Найденные группы сохраняются в базу и отправляются вам

#### Формат Excel-файла:

| Колонка    | Описание                     |
|------------|------------------------------|
| Название   | Имя группы/канала            |
| Юзернейм   | @username                    |
| Описание   | Описание группы              |
| Участники  | Количество участников        |
| Категория  | Категория (определяется ИИ)  |
| Тип        | channel / group / supergroup |
| Активность | active / inactive / unknown  |
| Ссылка     | https://t.me/...             |

### 6. Административные функции

> 👑 Доступно только для администраторов

1. Нажмите **"👑 Панель администратора"** в главном меню
2. Доступные функции:
    - **"📄 Получить лог файл"** — загрузка журнала событий
    - **"🔄 Актуализация базы данных"** — обновление типов и ID групп/каналов
    - **"🔍 Проверка групп"** — проверка групп на валидность
    - **"👤 Проверка аккаунтов"** — проверка подключенных аккаунтов

## 🟢 Определение активности групп

Бот автоматически определяет активность найденных групп:

| Статус        | Значение      | Описание                            |
|---------------|---------------|-------------------------------------|
| 🟢 `active`   | Активна       | Последнее сообщение ≤ 30 дней назад |
| 🔴 `inactive` | Неактивна     | Последнее сообщение > 30 дней назад |
| ⚪ `unknown`   | Не определено | Не удалось получить дату            |

## 🧑‍💻 Разработка и поддержка

### Модели базы данных (`database/database.py`):

- **Account** — системные аккаунты бота
- **UserAccountsTable** — аккаунты пользователей (StringSession)
- **Groups** — группы пользователей для отслеживания
- **TelegramGroup** — общая база найденных групп и каналов

### Основные модули:

- **AI-поиск:** `ai/ai.py` — `category_assignment()`, `get_groq_response()`, `search_groups_in_telegram()`
- **Парсинг:** `account_manager/parser.py` — обработка сообщений
- **Подписка:** `account_manager/subscription.py` — подписка на группы
- **Авторизация:** `account_manager/auth.py` — `CheckingAccountsValidity`

### Локализация:

Все строки интерфейса локализуемы (поддержка i18n в `locales/locales.py`).
