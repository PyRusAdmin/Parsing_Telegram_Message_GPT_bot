### 📡 Telegram бот для отслеживания ключевых слов в группах и каналах (https://t.me/AutoParseAlertBot)

- Версия: 0.0.6
- Дата выхода: 11 января 2026 года

Бот анализирует новые сообщения в указанных Telegram-группах и каналах, проверяя их на наличие заданных ключевых слов,
а также ищет с помощью ИИ (Groq AI) группы и каналы Telegram с последующей выдачей пользователю бота в личку.

Особенности:

- Поддержка мультиаккаунтности
- Экспорт данных в Excel
- Административная панель с актуализацией базы данных
- Многоязычный интерфейс (русский/английский)

## 📝 Доступные функции

- Отслеживание ключевых слов в указанных Telegram-группах и каналах
- Поиск групп/каналов по ключевым словам с помощью ИИ (Groq AI)
- Пересылка сообщений с ключевыми словами в указанную группу
- Управление аккаунтами Telegram (подключение через .session файлы)
- Получение логов работы бота через кнопку в панели администратора
- Экспорт данных в Excel (ключевые слова, каналы, полная база)
- 🔄 Актуализация базы данных (обновление ID и типа групп/каналов)
- Многоязычная поддержка (русский/английский)
- 📥 Массовое добавление групп

### 📥 Массовое добавление групп

Пользователи могут добавлять несколько групп и каналов одновременно через загрузку текстового файла:

- Формат файла: <b>.txt</b>
- Каждая строка должна содержать один username в формате <b>@username</b>
- Доступно через меню: <b>Настройки → 🔁 Обновить список</b>
- Бот автоматически обработает файл и добавит новые группы в список отслеживания

📌 Пример содержимого файла:
@group1
@channel1
@my_community

✅ Преимущества:
- Быстрое добавление большого количества каналов
- Удобная интеграция с внешними источниками
- Защита от дубликатов (система пропускает уже добавленные группы)

## ⚙️ Установка

1. Установите зависимости:
   ```bash
   python setup.py
   ```
   Или вручную:
   ```bash
   pip install -r requirements.txt
   ```

2. Создайте и заполните `.env` файл:
   ```text
   BOT_TOKEN=ваш_токен_бота
   ID=ваш_id_приложения
   HASH=ваш_hash_приложения
   ```

   Где:
    - `BOT_TOKEN` — токен вашего Telegram-бота (получается у [@BotFather](https://t.me/BotFather))
    - `ID` — API ID приложения Telegram
    - `HASH` — API HASH приложения Telegram

3. Запустите бота:
   ```bash
   python main.py
   ```
